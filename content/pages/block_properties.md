{{query((property type book))}}
        * Or, we can use the [[Advanced Queries]] :
created_at:: 1609337897497
last_modified_at:: 1609337897497
``` clojure
#+BEGIN_QUERY
{:title [:h2 "My books"]
:query [:find (pull ?b [*])
:where
[?b :block/properties ?p]
[(get ?p "type") ?t]
[(= "[[book]]" ?t)]]
}
#+END_QUERY
```
